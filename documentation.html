Quick Setup Guide - 5 Minutes to Success

Step 1: Install from Bubble Marketplace

📸 Screenshot: MapFlow Pro installation from Bubble marketplace 👆 Click to view
  1. Go to Bubble Marketplace
  2. Search "MapFlow Pro"
  3. Click "Install"

Step 2: Get FREE HERE API Key

Create your account and get API key:

👉 https://account.here.com/sign-in

  1. Sign up for HERE account
  2. Create project in HERE Developer Portal
  3. Generate API key
  4. Copy the key for Bubble configuration

Note: HERE offers a significant free tier (check current limits and conditions on HERE portal).

📸 Screenshot: HERE account creation and API key generation 👆 Click to view

Step 3: Configure MapFlow Element

📸 Screenshot: MapFlow element configuration with HERE API key 👆 Click to view
  1. Drop MapFlow element on your page
  2. Paste HERE API key in settings
  3. Configure your data source
  4. Set marker data fields

Important Layout Configuration

CRITICAL: For proper MapFlow functionality, follow this layout structure:
📸 Screenshot: Correct layout configuration - Column group with fixed height for MapFlow 👆 Click to view

Place MapFlow in a Column Group with:

  • Make this element fixed-height: ON (minimum 600-800px)
  • Fit height to content: OFF
  • Make this element fixed-width: OFF (for 100% responsive width)

Layout Structure:

Page Layout: Can use Row for sections (header, content, etc.) └── Column Group (wrapper for map) ├── Make this element fixed-height: ON ├── Min height: 600-800px ├── Fit height to content: OFF ├── Make this element fixed-width: OFF └── MapFlow Element (inside this column)

Note: Other page sections (headers, panels) can use Row layout, but the map container must be a Column with fixed height for optimal performance.

Step 4: Configure Geographic Address Inputs

Use Bubble's Native Geographic Address Input:

  1. Add two Input elements on your page:
    • Input "From": Content format → "Address"
    • Input "To": Content format → "Address"
  2. Configure draw_route workflow:
    Action: draw_route - Origin latitude: Input From's value's latitude - Origin longitude: Input From's value's longitude - Destination latitude: Input To's value's latitude - Destination longitude: Input To's value's longitude - Mode: car/walk/bike/transit

This enables users to enter any address using Bubble's native geographic address functionality with auto-suggestions.

Step 5: Add Pricing Configuration

For compute_fare Action:

  • Dynamic Values: Fully functional for database-driven pricing
  • Fixed Values: Recommended for taxi/VTC apps with standard rates

Example Configuration:

Action: compute_fare - base_fee: 3.50 (fixed value for standard rate) - per_km: 1.45 (fixed value for standard rate) - surge_multiplier: 1.0 (fixed, or connect to dynamic surge data)

Note: Create custom fields according to your business needs. These examples can be adapted to your specific data structure and requirements.

Advanced Pricing Configuration

For production apps, you can leverage additional pricing fields:

  • tier2_threshold_km: Set distance threshold for long-distance rates
  • tier2_per_km: Different rate for distances above threshold
  • free_km: Initial kilometers included in base fee
  • min_fare: Minimum charge regardless of distance
  • round_trip: Boolean for return journey calculations

Optional Fields Usage Examples:

  • per_minute: Add time-based pricing for traffic delays
  • surge_multiplier: Peak hour or demand-based pricing
  • distance_km_override: Manual distance correction if needed

These fields allow flexible pricing models adapted to your specific business requirements.

🗺️ HERE Geocoding Integration

Note: MapFlow does not include geocoding itself. Use Bubble's native 'Geographic address' input or HERE Geocoding API (via API Connector).

Setup (5 minutes)

Bubble API Connector Configuration:

  • API Name: HERE Geocoding
  • URL: https://geocode.search.hereapi.com/v1/geocode
  • Method: GET
  • Parameters:
    • q: [Input Address's value]
    • apiKey: [Your HERE API Key]
    • limit: 5

Workflow Example:

  1. Event: Input Address's value changed
  2. Action: External API Call → HERE Geocoding
  3. Result: Set origin_lat = Result's latitude
  4. Result: Set origin_lng = Result's longitude

User Experience:

  • User types: "Paris Gare de Lyon"
  • Auto-converts to: lat: 48.8449, lng: 2.3737
  • MapFlow uses coordinates automatically
